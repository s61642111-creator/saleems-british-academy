# 🇬🇧 Saleem's British English Academy

A full-stack British English learning app built with:
- **React 19** + **TypeScript** + **Vite 7**
- **Tailwind CSS 4** + **shadcn/ui** (New York)
- **tRPC v11** + **Drizzle ORM** + **MySQL**
- **Framer Motion** animations
- **wouter** routing
- Bilingual UI (English / Arabic)

## Getting Started

```bash
npm install
npm run dev
```

## Features
- 📖 6 structured lessons with quizzes
- 💬 Flashcard vocabulary system
- ⚡ Interactive practice exercises
- 🗺️ Journey / mistake tracker
- 📅 Weekly study plan
- 🏅 Badges & gamification
- 🌙 Dark mode + RTL Arabic support
